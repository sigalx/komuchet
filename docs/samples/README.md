# Образцы Документов

Папка содержит обезличенные или синтетические образцы документов текущего ручного процесса.

## Файлы

- `electricity-statement-anonymized.txt` - текстовый обезличенный образец структуры акта сверки и квитанции.
- `electricity-statement-anonymized.pdf` - PDF-версия того же обезличенного образца.
- `electricity-statement-redacted.pdf` - производная копия реального PDF, где ФИО и участок заменены тестовыми значениями, а QR-код удален.

## Приватные Артефакты

Оригинальные документы с персональными данными не должны храниться в репозитории. Для документации использовать только синтетические, обезличенные или redacted-копии.
